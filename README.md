# Fraud Detection Project

## Overview
End-to-end fraud detection pipeline built on real-world credit card transaction data.
Binary classification: fraudulent (1) vs legitimate (0) transactions.

## Dataset
Download from Kaggle: https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

Place the file at: `data/raw/creditcard.csv`

## Stack
Python, pandas, scikit-learn, XGBoost, LightGBM, SHAP, FastAPI
